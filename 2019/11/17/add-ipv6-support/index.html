基于第三方CDN的IPv6支持改造实施方案

发表于 2019-11-17 更新于 2020-05-03 分类于 系统运维

为响应国家号召，我们又需要给生产系统增加IPv6支持，而该工作同样可借助（支持IPv6的）CDN来完成。本文为IPv6改造工作的实施方案。

工作内容

  1. 应用程序改造
    • 排查程序中涉及IP地址的代码，修改不适用于IPv6地址的逻辑。
    • 排查数据库中涉及IP地址的字段，如果长度不够，需扩到39或以上。
  2. 按前文方案给应用系统套CDN。

（完）
